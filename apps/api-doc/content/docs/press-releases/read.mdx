import { Tab, Tabs } from 'fumadocs-ui/components/tabs';

Untuk mendapatkan siaran media berdasarkan ID, panggil titik akhir `<SERVER_URL>/webhook/press-releases/<ID>` menggunakan permintaan `GET`. Gantikan `<ID>` dengan pengecam unik siaran media yang ingin anda dapatkan.

<Tabs items={["cURL", "Python"]}>
  <Tab value="cURL">
    ```bash
    curl -X GET "<SERVER_URL>/webhook/press-releases/<ID>" \
      -H "Authorization: Bearer <TOKEN_AKSES_ANDA>"
    ```
  </Tab>
  <Tab value="Python">
    ```python
    import requests

    response = requests.get(
        "<SERVER_URL>/webhook/press-releases/<ID>",
        headers={
            "Authorization": "Bearer <TOKEN_AKSES_ANDA>"
        }
    )
    ```
  </Tab>
</Tabs>

## Respons Ralat

### Siaran Media Tidak Dijumpai

Jika siaran media dengan `<ID>` yang dinyatakan tidak wujud, anda akan menerima respons `404 Tidak Dijumpai`.

```bash
HTTP/1.1 404 Not Found
Content-Type: application/json

{
  "code": "not_found",
  "message": "Not Found"
}
```

Ini menunjukkan bahawa anda perlu mengesahkan `<ID>` yang diberikan dan memastikan bahawa siaran media itu wujud.
